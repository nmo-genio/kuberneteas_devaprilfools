# 🍵 KuberneTEAS: Enterprise-Grade Tea Orchestration

Welcome to **KuberneTEAS**, the world's first (and hopefully last) SRE dashboard dedicated to the high-availability orchestration of Camellia Sinensis. This project was built for the **DEV April Fools' Challenge**, parodying Kubernetes, Prometheus, and the legendary **RFC 2324 (HTCPCP/1.1)**.

## 🚀 Features

### 1. 📊 Prom-TEA-us
A full-featured monitoring suite for your tea infrastructure.
- **Graph Tab:** Real-time metrics for `earl_grey`, `chamomile`, and `oolong` saturation.
- **Alerts Tab:** Critical alerts like `TeaIsCold` and `TooMuchSugar`.
- **Status Tab:** Runtime info for the "Tea Storage Data Base" (TSDB).
- **Help Tab:** A Prom-TEA-QL cheat sheet for advanced querying.

### 2. 🗺️ Pod Topology
Visualize the physical distribution of your tea leaves across the cluster.
- **Node Taints:** Beware of the `coffee-smell` taint which prevents scheduling.
- **Pod Status:** Real-time tracking of `Steeping`, `Running`, and `CrashLoopBackOff` states.
- **Tea-Mesh:** A visualization of the inter-mug communication protocol (IMCP).

### 3. 🕸️ Aroma-Mesh Control Plane
Manage inter-service communication with the Aroma-Mesh.
- **mTLS:** Mutual Tea Leaf Security (ENFORCED).
- **Service Telemetry:** Monitor traffic splitting between `Earl Grey` and `English Breakfast`.
- **Honey-Pot Service:** Critical anomaly detection for honey crystallization loops.

### 4. 🐚 Kettle Shell (K-SH)
A retro-style terminal for low-level infrastructure management.
- **RFC 2324 Compliance:** Every command returns a **`HTTP 418: I'm a teapot`** error.
- **Restricted Access:** Only `schedule coffee pods` operations are (unsuccessfully) permitted.

### 5. 💧 Water Ingress Controller
Manage the flow of H2O into your mugs.
- **Ingress Rules:** Block unauthorized `coffee.pot` traffic to the `null-sink`.
- **Auto-Flush Operator:** Automatically purges stagnant bergamot particles.

### 6. 🌡️ Heat Operators
Reconcile the thermal state of your substrate.
- **Thermal Reconciliation:** Target temperature of 98.5°C for optimal steeping.
- **Safety Interlocks:** Checks for "Water Presence" and "Coffee Absence" before boiling.

### 7. 🌬️ Aroma Sidecar Injector
Inject essential flavors into your tea pods.
- **Bergamot Saturation:** Real-time tracking of aroma propagation.
- **Honey Injector:** Beware of sidecar crashes due to glucose crystallization.

### 8. 📖 Runbook (SOP-418)
Standard Operating Procedures for the modern Tea Reliability Engineer.
- **Triage:** Step-by-step guides for "Cold-Tea-Syndrome".
- **Nuclear Option:** If all else fails, re-provision the kitchen namespace and go to the pub.

### 9. ⚠️ Teapot Error (RFC 2324)
A fatal exception modal triggered by unauthorized `COFFEE.EXE` deployments.
- **Compliance:** Strictly enforces the "I am a teapot" status code.

## 🛠️ Technical Details

- **Framework:** React 18 + Vite + TypeScript
- **Styling:** Tailwind CSS (Dark Theme / Black Background)
- **Animations:** Framer Motion (`motion/react`)
- **Icons:** Lucide React
- **Charts:** Recharts
- **AI:** Gemini API (via `@google/genai`) for nonsensical Root Cause Analysis.

## 🎭 Easter Eggs

- **Scaling Effect:** Scaling tea replicas triggers a "boiling" sound effect and a CSS "steam" blur filter.
- **Taints:** Tooltips on "Chai Pods" reveal hidden `coffee-smell` taints.
- **Sidecars:** Logs frequently mention `Honey-Injector` sidecars in `CrashLoopBackOff` due to crystallized honey.
